Configuration

Pour que l'application fonctionne correctement, assurez-vous, avant tout, de faire les éléments suivants:

Lire le fichier de l’auteur

  • Allez vers Fichier > Lire > Auteur ou appuyez sur ctl+shift+a

  • Sélectionnez le fichier .txt de l'auteur dans le dossier de l'auteur

  • Notez que le nom du fichier de l'auteur avec son chemin est ajouté au champ de texte appelé Auteur dans la fenêtre principale

Lire le fichier du livre

  • Allez vers Fichier > Lire > Livre ou appuyez sur ctl+shift+l

  • Sélectionnez le fichier .txt du livre dans le dossier du livre

  • Notez que le nom du fichier du livre avec son chemin est ajouté au champ de texte appelé Livre dans la fenêtre principale

Appliquer les fichiers sélectionnés

  • Le bouton Appliquer est désactivé par défaut et ne peut être activé que si les fichiers de l’auteur et du livre ont été sélectionnés

  • Dès que les fichiers de l’auteur et du livre ont été sélectionnés, appuyez sur le bouton Appliquer

  • Le texte Appliquer change à OK ce qui signifie que les fichiers de l’auteur et du livre ont été lu avec succès

Application

Cette application peut ajouter des auteurs et des livres à la base de données, ainsi que rechercher un auteur et énumérer ses oeuvres et rechercher un livre et afficher ses informations.

Il peut également montrer un rapport aux auteurs dans la base de données ainsi qu'un rapport pour les livres.

Ajouter un auteur

  • Allez vers Application > Ajouter > un auteur ou appuyez sur alt+a

 

  • Entrez les informations de l'auteur que vous souhaitez ajouter dans les champs de texte dans la boîte de dialogue Ajouter un auteur et appuyez sur le bouton Ajouter

  • [REMARQUE]

    • Les champs de texte ne faut pas être vide

    • Le champs Code doit être entier

    • Pas de doublon pour le code de l’auteur

    • Pas de doublon pour le nom de l’auteur

Ajouter un livre

  • Allez vers Application > Ajouter > un livre ou appuyez sur alt+l

  • Entrez les informations du livre que vous souhaitez ajouter dans les champs de texte dans la boîte de dialogue Ajouter un livre et appuyez sur le bouton Ajouter

  • [REMARQUE]

    • Les champs de texte ne faut pas être vide

    • Le champs Code doit être entier

    • Le champs Code de l’auteur doit être entier

    • Le champs Prix doit être entier

    • Le champs Nombre de pages doit être entier

    • Pas de doublon pour le code du livre

    • Pas de doublon pour le nom de l’auteur

    • Le code de l’auteur doit exister dans la base de donnée

Chercher un auteur

  • Allez vers Application > Chercher > un auteur ou appuyez sur ctl+a

  • Entrez le nom ou le code de l'auteur que vous souhaitez chercher dans le champ de texte du dialogue Chercher un auteur et appuyez sur Chercher

  • Si l'auteur a été trouvé, les oeuvres de l'auteur apparaîtraient dans un dialogue Oeuvres de l’auteur

  • Si l’auteur n’a pas été trouvé, un dialogue Attention apparaîtrait

 

Chercher un livre

  • Allez vers Application > Chercher > un livre ou appuyez sur ctl+l

  • Entrez le nom ou le code du livre que vous souhaitez chercher dans le champ de texte du dialogue Chercher un livre et appuyez sur Chercher

 

  • Si le livre a été trouvé, les informations du livre apparaîtraient dans un dialogue Info

  • Si le livre n’a pas été trouvé, un dialogue Attention apparaîtrait

Rapport par auteur

  • Allez vers Application > Rapport > par auteur

  • Le rapport par auteur apparaîtra dans la boîte de dialogue Rapport par auteur

Rapport par livre

  • Allez vers Application > Rapport > par livre

 

  • Le rapport par livre apparaîtra dans la boîte de dialogue Rapport par livre
